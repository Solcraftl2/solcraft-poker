Endpoint Disponibili

Informazioni API

GET /api

Restituisce informazioni di base sull'API, inclusa la versione e lo stato.

Elenco Tornei

GET /api/tournaments

Restituisce l'elenco di tutti i tornei disponibili.

Dettagli Torneo

GET /api/tournaments/:id

Restituisce i dettagli di un torneo specifico, inclusi gli investimenti associati.

Creazione Torneo

POST /api/tournaments

Crea un nuovo torneo con i dettagli forniti.

Creazione Investimento

POST /api/investments

Crea un nuovo investimento per un torneo specifico.

Registrazione Utente

POST /api/users/register

Registra un nuovo utente nel sistema.

Login Utente

POST /api/users/login

Autentica un utente e restituisce un token JWT.
